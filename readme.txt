== Description ==
## NOW SUPPORTS WINDOWS PHONE ##

Based on [Matthias Reuter Plugin](http://wordpress.org/extend/plugins/mobile-device-detect/). This Wordpress plugin integrates the Detect Mobile Browsers (Open source mobile phone detection) script from [http://detectmobilebrowsers.com/](http://detectmobilebrowsers.com/) to wordpress.

It allows to redirect mobile visitors to a custom target URL. Mobile Device Detect for Wordpress is licensed to GNU General Public License v3. **Now includes the Detect Mobile Browser was created by [Chad Smith](http://twitter.com/chadsmith).**

More Informations, Installation Guide and Download are available on the plugin homepage of [Mobile Device Detect Reloaded for Wordpress](http://www.desimon.net/mobile-device-detect-reloaded).


Follow me on Twitter to keep up with the latest updates [Guilherme Desimon](http://twitter.com/guidesimon/)

== Installation ==

Simple install:

1. Upload `loadMobileDeviceDetectReloaded.php` and `mobile_device_detect.php` to the `/wp-content/plugins/mobile_device_detect_reloaded/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure in Mobile Device Detect Reloaded tab in Admin Menu.
